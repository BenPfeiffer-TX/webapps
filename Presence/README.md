## Running the Application 🎉
------------------------

To run the application, simply execute:

```bash
git clone https://github.com/BenPfeiffer-TX/webapps
cd webapps/TAAG
go build . && ./main
```

Then, navigate to `http://localhost:8080` in your web browser to access the status board 🔗.
